SportPulse is currently in the early SEO and AdSense-readiness phase.

The current goal is not to add expensive features, live sports APIs, fantasy tools, or betting/market data.

The current goal is:

1. Let Google crawl the site.

4. Wait for early indexing signals.
5. Prepare for a responsible AdSense application after initial indexation.

## Monitoring schedule

### Check 1 — 2 to 3 days after sitemap submission

Review in Google Search Console:

- Pages report
- Sitemaps report
- URL Inspection for homepage
- URL Inspection for key hub pages
- Crawling or indexing errors

Questions to answer:

- Is the sitemap still successful?
- Are pages being discovered?
- Are any pages indexed?
- Are any important pages excluded?
- Are there errors such as 404, redirect issues, server errors, blocked by robots.txt, or duplicate/canonical issues?
- Does Google show “Discovered - currently not indexed”?
- Does Google show “Crawled - currently not indexed”?

Notes:

“Discovered - currently not indexed” or “Crawled - currently not indexed” can be normal for a new site. Do not panic immediately. Watch whether the situation improves over time.

Review:

- Excluded pages

- Whether Google has started understanding the site structure

Decision after Check 3:

If pages are being indexed and no major errors appear, SportPulse can move toward the first AdSense application.

If many pages are discovered but not indexed, review article quality, internal linking, uniqueness, and page usefulness before applying.

If technical errors appear, fix them before applying.

## AdSense readiness checkpoints

Before applying for AdSense, confirm:

- Homepage works
- Sitemap works
- robots.txt works
- Search Console property is verified
- Sitemap is successfully submitted
- Important pages are indexable
- At least some pages are indexed
- No important page returns 404
- No placeholder content exists
- No lorem ipsum exists
- Footer links work
- Breadcrumbs work on article pages
- About page is complete
- Contact page is complete
- Privacy Policy is complete
- Terms of Use are complete
- Legal Notice / Impressum is complete
- Site does not claim live data it does not have
- Site does not use copyrighted team or league logos
- Site does not offer betting tips or gambling advice
- Site clearly states that SportPulse is independent

## Current no-go rules

Do not add these before traffic validation:

- paid sports data API
- paid betting odds API
- fantasy sports API
- user accounts
- database
- paid SEO tools
- expensive AI workflows
- complex backend features
